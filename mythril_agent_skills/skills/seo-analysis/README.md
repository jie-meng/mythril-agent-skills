# SEO Analysis

`seo-analysis` is a standalone SEO audit skill for AI coding assistants.

It helps with:

- technical SEO reviews
- indexing and crawlability checks
- title/meta and schema audits
- content-gap and cannibalization analysis
- page-speed and Core Web Vitals triage

The workflow is adapted from the public SEO audit skill in
[`nowork-studio/toprank`](https://github.com/nowork-studio/toprank), but
trimmed into a self-contained skill format that fits the
`mythril-agent-skills` repository structure.
