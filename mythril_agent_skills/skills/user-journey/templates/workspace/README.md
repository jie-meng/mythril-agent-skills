# {{TITLE}}

A user-journey workspace generated by the [`user-journey`](https://github.com/jie-meng/mythril-agent-skills) skill.

## Contents

| File | What it is |
|---|---|
| `JOURNEY.md` | Business requirements, persona, mermaid flow, narrative — for humans |
| `journey.json` | Structured stages & steps — for the renderer |
| `DESIGN.md` | Visual style ([Google DESIGN.md format](https://github.com/google-labs-code/design.md)) |
| `index.html` | Self-contained preview — open in a browser |
| `assets/` | CSS + JS for the renderer (do not hand-edit) |
| `preview.py` | Local HTTP fallback if double-click doesn't work |

## How to view

**Double-click `index.html`** — opens in your default browser. Works offline.

If the page is blank, your browser is blocking `file://` fetches:

```bash
python3 preview.py
```

Opens [http://localhost:8765](http://localhost:8765) automatically. Press `Ctrl+C` to stop. Requires only Python 3.10+.

## Three views

| View | Shortcut | Use for |
|---|---|---|
| **Map** | `M` | Overview of all stages — pan/zoom with mouse, `+`/`-`/`0` keys |
| **Stage** | `S` | Drill into one stage — `←` `→` to switch stages |
| **Presenter** | `P` | Full-screen demo — `←` `→` to advance, `B` to blank, `Esc` to exit |

You can deep-link to a specific view, e.g. `index.html#present/sign-up` jumps straight into presenter mode on the "sign-up" stage.

## Editing the journey

Ask your AI assistant — invoke the `user-journey` skill and give it this directory path. It reads all three files and continues with full context.

If you edit `JOURNEY.md` or `journey.json` by hand, run:

```bash
python3 SKILL_PATH/scripts/validate_sync.py .
```

to confirm the two files stay in sync.

## Changing visual style

Edit `DESIGN.md` directly. The renderer reads it on every page load. To swap to a different preset, copy a different preset from `SKILL_PATH/templates/design-styles/` over `DESIGN.md` and refresh the browser.

## Sharing

Zip the entire directory and send it. Recipient unzips and double-clicks `index.html`. No install needed.

For a public link, commit to a git repo with GitHub Pages enabled — `index.html` works as-is.
